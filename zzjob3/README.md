# 第三次上机作业 - 大提琴社招新网站

## 项目文件说明

### 文件介绍
1. **job3.html** - 主页面
2. **club_intro.html** - 社团详细介绍页面
3. **README.md** - 项目说明文档
4. **cello_club_music.mp3** - 社团音乐文件
5. **cello_club_img.png** - 社团照片文件
6. **cello_club_video.mp4** - 社团视频文件
## 功能实现说明

### 1. 背景音乐播放功能 ✅
- 左上角音乐播放按钮
- 点击切换播放/暂停状态

### 2. 导航链接和内容 ✅
- 五对内部链接：社团简介、活动安排、风采展示、社员故事、在线报名
- 每个部分都有返回顶部按钮
- 使用AI生成了1000字以上内容

### 3. 图片和跳转功能 ✅
- 插入社团活动图片
- 点击图片跳转到club_intro.html
- 返回链接功能完善

### 4. 视频导入功能 ✅
- 使用embed标签导入MP4视频
- 响应式设计，适配不同屏幕

### 5. 在线报名表单 ✅
- 10个字段的完整报名表
- 使用table布局
- 包含所有要求的字段类型

## 技术特点

- 响应式设计，适配各种设备
- 现代化UI设计，美观大方
- 完整的交互功能
- 符合Web标准
- 代码结构清晰，易于维护
